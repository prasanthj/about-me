ABOUT ME

I am pursuing Masters in Computer Science and Engineering at The Ohio State University, Columbus. Currently working as a Graduate Research Assistant (GRA) under Prof. Arnab Nandi focussing on large scale data analytics and big data interaction research. I am also a Google Summer of Code GSoC 2012 student working on providing CUBE operation support in Apache Pig. During spring quarter, I worked for Advanced Computing Center for the Arts and Design (ACCAD) as a GRA under Prof. Alan Price on real time data streaming using VRPN. I also worked as a web application developer at OSUOGG.

Prior to that, I worked as a Software Engineer at Samsung India Software R&D Center for 3 years as a part of Game and Connectivity team. I completed Bachelor of Engineering (Hons) in Computer Science from Birla Institute of Technology and Science,Pilani,Goa. I also worked as an intern for 6 months in SAP Labs, Bangalore as a part of NetWeaver User Interface Foundation Team.

    • Pig Latin mode for CodeMirror: In this project I implemented pig mode in CodeMirror(online javascript based code editor) with support for syntax highlighting, autosuggestion and autocompletion for Pig Latin language (used by Apache Pig).

    Map-side fragment replicated join in YSmart: As a part of large scale distributed systems course project, I implemented map-side fragment replicated join support for YSmart. In this the smallest table in the join query is distributed to all mappers using hadoop's distributed cache and an in-memory map side join is performed thereby reducing the amount of intermediate data transfer in shuffle phase.
